Бухгалтерія для України, редакція 2.0. (2.0.8.3)
Бухгалтерія для України, редакція 1.2. (1.2.44.2)
Управление торговлей для Украины, редакция 3.1 (3.1.6.1) 
Управление небольшой фирмой для Украины, редакция 1.6 (1.6.5.2) 


Для конфигураций, ниже по списку, модуль работает с некоторыми ограничениями, ввиду того что конфигурации устарели и не обладают рядом современных методов в работе.
Управление торговлей для Украины, редакция 2.3. редакция (2.3.29.1)
Управление торговым предприятием для Украины, редакция 1.2. (1.2.45.2)

Файлы из 1С добавляются следующим образом:
1. Вы выбираете документ
2. отображается печатная форма документа, необходимо сохранить ее через "Файл - сохранить как" и выбрать название и сам файл
3. После этого в список документов добавится документ для передачи в сервис Подпиши.Онлайн, нажмите на документе правой кнопкой мыши и кнопку "ПРикрепить файл" и выбирите ранее сохраненный файл.
4. После этого вы можете отправить документ на подпись и подписать его.
